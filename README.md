# Wibu

Wibu is a Chrome extension that helps users translate highlighted or manually typed web text through a simple side panel interface.

It was built to make online reading easier for students, language learners, newcomers, and anyone who needs quick translation support while browsing the web.

## Features

* Translate highlighted web text through the right-click menu
* Manually type or paste text for translation
* Choose source and target languages with flag-based language selectors
* Save recent translation logs locally in the browser
* Delete individual logs or clear all logs
* Save language preferences locally
* Built-in tutorial showing how to use Wibu
* Simple privacy-conscious interface with no account required

## How It Works

Users can highlight text on a webpage, right-click, and choose **Ask Wibu**. Wibu opens in Chrome’s side panel, where the user can choose languages and translate the selected text.

Users can also type or paste text directly into Wibu and translate it manually.

## Privacy

Wibu does not sell user data, use tracking, or use analytics.

Translation logs and language preferences are saved locally in the user’s browser. When the user clicks **Translate**, the selected or typed text is sent to MyMemory’s translation API to return a translation.
